Welcome
=======

Welcome to the PAN-OS Ansible Lab!

This lab will give you hands-on experience using the PAN-OS Ansible modules.
You will use the modules to automate configuration changes, operational
commands, and software upgrades on a PAN-OS firewall, showing you different
possibilities for managing devices in your own environment.
